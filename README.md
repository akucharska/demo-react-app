## Local instalation

```npm install```

## Run app on phone
1. Install `expo client` on your phone.
2. Go to `https://exp.host/@agata.kucharska/demo-react-app`.
3. Scan QR Code with your phone. It will run application on your phone.


## Run app in the browser
1. Go to: `https://exp.host/@agata.kucharska/demo-react-app`.
2. Click `Open project in the browser` button, ans then `Open project`.
3. Click `Tap to play` and `Open with Chrome`.
4. Scroll down nad click `Open project using Expo`.
